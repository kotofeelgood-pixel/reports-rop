# Анализ конкурентов

## 1. Qolio - проверка звонков и чатов

**Разработчик:** dealapp.io  
**Установок:** 768  
**Модель:** Подписка, встроенные покупки

### Ключевые особенности:
- **Контроль качества звонков** — проверка, оценка и анализ звонков по чек-листам
- **Управление коммуникациями** — формирование списков для проверки с фильтрацией
- **Комментирование** — возможность комментировать звонки во время проверки, к отдельным участкам или критериям
- **Теги и флажки** — система маркировки для наглядности и быстрого поиска
- **Библиотека звонков** — база знаний с наглядными кейсами
- **Автоматические уведомления** — оповещения операторов об оценке звонков
- **Обратная связь** — двусторонняя коммуникация между операторами и менеджерами
- **Аналитика по критериям** — статистика и динамика по различным метрикам качества
- **Визуализация результатов** — цветовая индикация (красный/жёлтый/зелёный) для быстрой оценки

### Преимущества:
- Ускорение процесса проверки в 2 раза
- Увеличение объёма проверок на 40%
- Автоматизация отчётов
- Единое рабочее пространство для контроля качества

### Слабые стороны:
- Меньше внимания к финансовым метрикам (стоимость звонков)
- Ограниченная аналитика по продажам и сделкам
- Фокус на качестве, а не на количественной аналитике

## 2. BI-отчет: Аналитика звонков PRO

**Разработчик:** СкайВеб24 (SkyWeb24)  
**Установок:** 535  
**Модель:** Подписка

### Ключевые особенности:
- **17 уникальных графиков и сводных таблиц** — комплексная визуализация данных
- **Детальная фильтрация** — по дате, сотруднику, типу звонка, номеру портала
- **Ключевые показатели** — количество, длительность, стоимость звонков
- **Детальная таблица звонков** — полная информация: оператор, клиент, статус, тип
- **Хронология звонков** — динамика по месяцам с визуализацией
- **Анализ расходов** — отслеживание стоимости по операторам и подразделениям
- **Карта телефонной нагрузки** — распределение звонков по дням недели и часам
- **Таблица активности по клиентам** — детальная статистика по каждому клиенту
- **ТОП-списки** — рейтинги по количеству и стоимости звонков
- **Индивидуальные решения** — разработка кастомных BI-отчётов под ключ

### Преимущества:
- Максимально полная аналитика
- Гибкая настройка и адаптация под задачи
- Интеграция с BI Конструктором Bitrix24
- Фокус на финансовых метриках

### Слабые стороны:
- Нет функций контроля качества (чек-листы, оценка)
- Отсутствие обратной связи с операторами
- Нет библиотеки звонков для обучения

## 4. Детализация звонков

**Разработчик:** Bambit  
**Установок:** 1,020  
**Модель:** Подписка

### Ключевые особенности:
- **Детальная фильтрация по CRM сущностям** — фильтрация по лидам, компаниям, контактам с выбором конкретных номеров телефонов
- **Гибкая фильтрация** — по интервалу дат, номеру клиента, типу звонков (входящие/исходящие/все), подразделению, сотруднику
- **Группировка по сотрудникам** — возможность суммировать данные по сотрудникам подразделения или просматривать детализацию по каждому звонку
- **Визуализация данных** — круговые диаграммы по длительности и стоимости звонков
- **Итоговые показатели** — автоматический расчёт итогов по времени и стоимости разговоров
- **Детальная таблица звонков** — полная информация по каждому звонку: дата, переговорщик, номер клиента, длительность, стоимость
- **Печать отчётов** — встроенная функция печати отчётов
- **Экспорт в XLS** — скачивание отчётов в формате Excel
- **Настройка статусов** — выбор статусов вызовов, учитываемых при формировании отчётов
- **Права доступа** — интеграция с системой прав доступа Bitrix24

### Преимущества:
- Простота использования — понятный интерфейс с пошаговым формированием отчётов
- Глубокая интеграция с CRM — работа с лидами, компаниями, контактами
- Гибкая группировка данных — от детализации до агрегированных показателей
- Визуализация ключевых метрик — наглядные круговые диаграммы
- Функции печати и экспорта — удобная работа с отчётами

### Слабые стороны:
- Ограниченная визуализация — только круговые диаграммы, нет других типов графиков
- Нет динамики и трендов — отсутствие графиков изменения показателей во времени
- Нет аналитики по продажам — отсутствие связи со сделками и конверсией
- Нет контроля качества — отсутствие функций оценки и комментирования звонков
- Статичные отчёты — нет интерактивных дашбордов и настраиваемых панелей

## 5. Мои Звонки

**Разработчик:** Облако Технологий  
**Установок:** 22,742  
**Рейтинг:** 4/5  
**Модель:** Подписка, встроенные покупки  
**Категория:** Интеграции

### Ключевые особенности:
- **Интеграция Android смартфонов с Bitrix24** — мобильное приложение для синхронизации звонков
- **Автоматический набор номера** — возможность нажать на номер в Bitrix24 и автоматически начать звонок через телефон
- **Автоматическое прикрепление звонков** — звонки автоматически привязываются к карточке клиента или сделке в CRM
- **Автоматическое создание лидов/сделок** — при звонке с неизвестного номера автоматически создаётся лид или сделка
- **Аудио записи разговоров** — возможность прослушивания и скачивания записей звонков (зависит от модели телефона)
- **Всплывающие уведомления** — при входящем звонке отображается имя клиента из Bitrix24
- **Отправка SMS** — возможность отправки SMS из интерфейса Bitrix24 через Android смартфон
- **Детальная информация о звонках** — отображение типа звонка, длительности, статуса (успешный/отклонён), времени
- **Связь с контактами** — автоматическое определение контакта по номеру телефона

### Преимущества:
- Удобная мобильная интеграция — работа со звонками прямо из Bitrix24
- Автоматизация процессов — автоматическое создание и привязка сущностей CRM
- Большая база пользователей — 22K+ установок с высоким рейтингом (4/5)
- Экономия времени — не нужно вручную искать и набирать номера
- Записи разговоров — возможность прослушивания для контроля качества

### Слабые стороны:
- Ограниченная платформа — только Android, нет поддержки iOS
- Зависимость от устройства — функция записи звонков работает не на всех моделях
- Нет веб-аналитики — фокус на мобильной интеграции, а не на аналитике звонков
- Нет визуализации данных — отсутствие графиков и дашбордов
- Нет групповой аналитики — отсутствие отчётов по отделам и сотрудникам
- Платные функции — часть функционала доступна только по подписке
