# Список недостающих компонентов

## Layout
- [ ] **App** - Обертка приложения для глобальных настроек
- [ ] **Container** - Контейнер для центрирования контента
- [ ] **SidebarLayout** - Layout с боковой панелью

## Element
- [ ] **Calendar** - Календарь для выбора дат
- [ ] **Card** - Карточка с заголовком, телом и футером
- [ ] **Chip** - Индикатор числа или состояния
- [ ] **Countdown** - Обратный отсчет с опциями управления
- [ ] **FieldGroup** - Группировка кнопко-подобных элементов
- [ ] **Kbd** - Элемент для отображения клавиатурного ввода
- [ ] **Skeleton** - Скелетон для загрузки контента

## Form
- [ ] **ColorPicker** - Выбор цвета
- [ ] **Form** - Компонент формы с валидацией
- [ ] **FormField** - Контейнер для элементов формы с валидацией
- [ ] **InputDate** - Поле ввода даты
- [ ] **InputTime** - Поле ввода времени

## Navigation
- [ ] **Link** - Обертка вокруг NuxtLink с дополнительными пропсами
- [ ] **NavigationMenu** - Список ссылок (горизонтальный или вертикальный)

## Overlay
- [ ] **DropdownMenu** - Контекстное меню действий
- [ ] **Popover** - Немодальное всплывающее окно
- [ ] **Slideover** - Диалог, выезжающий с любой стороны экрана
- [ ] **Tooltip** - Всплывающая подсказка при наведении

## Page
- [ ] **PageCard** - Предустановленная карточка с заголовком, описанием и ссылкой
- [ ] **PageColumns** - Адаптивная многоколоночная система макета
- [ ] **PageList** - Вертикальный список для контента

## Dashboard
- [ ] **DashboardGroup** - Контейнер дашборда с управлением состоянием сайдбара
- [ ] **DashboardSearch** - Готовый компонент Command Palette для дашборда
- [ ] **DashboardSearchButton** - Кнопка для открытия Dashboard Search

## AI Chat (2.2+)
- [ ] **ChatMessage** - Компонент сообщения чата с иконкой, аватаром и кнопками действий
- [ ] **ChatMessages** - Список сообщений чата для интеграции с Vercel AI SDK
- [ ] **ChatPalette** - Интерфейс чата для отображения в модальном окне
- [ ] **ChatPrompt** - Улучшенный Textarea для отправки промптов в AI чате
- [ ] **ChatPromptSubmit** - Кнопка отправки промпта с автоматической обработкой статуса

## Editor (2.2+)
- [ ] **Editor** - Редактор текста на TipTap (markdown, HTML, JSON)
- [ ] **NewEditor** - Новый редактор
- [ ] **NewEditorDragHandle** - Перетаскиваемый маркер для переупорядочивания блоков
- [ ] **NewEditorEmojiMenu** - Меню эмодзи при вводе символа `:`
- [ ] **NewEditorMentionMenu** - Меню упоминаний при вводе символа `@`
- [ ] **NewEditorSuggestionMenu** - Меню команд при вводе `/`
- [ ] **NewEditorToolbar** - Настраиваемая панель инструментов редактора

## Content
- [ ] **ContentSearch** - Готовый Command Palette для документации
- [ ] **ContentSearchButton** - Кнопка для открытия поиска контента
- [ ] **ContentSurround** - Кнопки "Предыдущая" и "Следующая" для навигации
- [ ] **ContentToc** - Липкая таблица содержимого с подсветкой активных разделов

## Color Mode
- [ ] **ColorModeAvatar** - Avatar с автоматической сменой изображения по теме
- [ ] **ColorModeButton** - Кнопка переключения светлой/темной темы
- [ ] **ColorModeImage** - Изображение с разными источниками для светлой/темной темы
- [ ] **ColorModeSelect** - Select для переключения между системной, темной и светлой темой

## i18n
- [ ] **LocaleSelect** - Select для переключения между локалями
